# git 使用说明

需要先在https://github.com/zgj12306下面先new repository 如：git

在本地目录下面git init初始化

echo "# git" >> README.md

git add README.md

git commit -m "first commit"

git remote add origin https://github.com/zgj12306/git.git

git push origin master

或：

git remote add origin https://github.com/zgj12306/git.git

git pull origin master
